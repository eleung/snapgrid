# Security Policy

## Supported versions

snapgrid is pre-1.0; security fixes are applied to the latest published minor.

| Version | Supported |
| --- | --- |
| latest | ✅ |
| older | ❌ |

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Instead, report it privately via [GitHub Security Advisories](https://github.com/eleung/snapgrid/security/advisories/new),
or email the maintainers. Include:

- a description of the issue and its impact,
- steps to reproduce (a minimal repro is ideal),
- affected versions.

You'll get an acknowledgement within a few days. Once the issue is confirmed and fixed, we'll
publish a release and credit you (unless you prefer to remain anonymous).
